# Context: Amen QR Blessing App
- **Goal:** A mobile-first web app for church members to receive a random encouraging verse after scanning a QR code.
- **Tech Stack:** - Frontend: React 19 (TypeScript, Vite, Tailwind CSS, Framer Motion).
    - Backend: .NET 10 Web API (Minimal APIs).
- **Latest Tech:** Use the React Compiler features (no need for manual useMemo/useCallback) and C# 14 features.
- **Primary User Flow:** Scan QR -> See Verse -> Click "Amen" -> See Success Message with a "Next QR" option.
